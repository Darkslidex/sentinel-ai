# 🛡️ Sentinel AI: Ecosistema de Seguridad 360°

**Sentinel AI** es una solución de seguridad híbrida que integra Inteligencia Artificial para la protección física y digital. Diseñado para entornos de alta exigencia como bancos o centros logísticos.

### 🚀 Módulos Principales
* **Vision Sentinel:** Motor de visión computacional (YOLOv8) con detección perimetral poligonal y captura de evidencia forense.
* **Anti-Fraud Engine:** Modelo predictivo optimizado con técnica SMOTE para el balanceo de transacciones bancarias.
* **Command Center:** Dashboard interactivo en tiempo real (Streamlit) con visualización asíncrona de eventos y simulación de ataques.

### 🛠️ Tech Stack
- **Python 3.10+**
- **IA:** Ultralytics YOLOv8, Scikit-learn (Random Forest), PyTorch 2.6.
- **Frontend:** Streamlit.
- **Procesamiento:** OpenCV, Pillow, Pandas, Numpy.

### 📈 Valor de Negocio (ROI)
El sistema cuantifica automáticamente el capital protegido basándose en la prevención de robos físicos y la mitigación de fraudes digitales detectados.
